
# Portfolio

Portfolio is my very simple personal website containing some of my main projects and my description through the skills I have acquired and what I am capable of doing. It is a simple portfolio where you can also download my resume and contact me in various ways. It is entirely written in [React](https://it.legacy.reactjs.org/docs/getting-started.html) and utilizes [Material UI](https://mui.com/material-ui/) (MUI) for the front-end.
