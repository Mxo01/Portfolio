
# Portfolio

My personal portfolio including my Tech Stack, my Experience, my Education and some of my projects. Here, you can also see my CV and download it! 

[Visit here 🧐](https://portfolio-34409.web.app/)
## Deployment

To deploy this project you need to build it first using:

```
ng build
```

And then you can deploy it with firebase using:

```
firebse deploy
```
